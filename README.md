# Stack Master Labs

Build your perfect burger — then find out what it actually costs.

You pick the bun, stack the protein, load up the toppings. It feels like a fun menu experience. Then you hit checkout, and the app shows you the land use, water, carbon, and animal toll behind what you just built. The environmental data is real (sourced from Poore & Nemecek 2018 and the FAO). The discomfort is intentional.

## Running it

You'll need Node.js installed.

```bash
npm install
npm run dev
```

## Stack

React 19 · TypeScript · Vite · Tailwind v4 · Motion/React
